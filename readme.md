Менеджер контактов на Backbone.js
=================================

Функциональность:
-----------------
* Добавление контактов (имя, skype, телефоны, фотография)
* Вывод информации о контакте
* Изменение всех параметров контакта
* Удаление контакта
* Поиск контакта

Используемые технологии:
------------------------
* backbone
* underscore
* jquery
* scss
* gulp
* bower

Сборка:
-------
* npm install gulp
* bower install
* gulp bower
* gulp css (gulp watch:css)
* gulp build
