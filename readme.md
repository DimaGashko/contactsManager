# Менеджер контактов на Backbone.js

Функциональность:
1. Добавление контактов (имя, skype, телефоны, фотография)
2. Вывод информации о контакте
3. Изменение всех параметров контакта
4. Удаление контакта
5. Поиск контакта

Используемые технологии:
1. backbone
2. underscore
3. jquery
4. scss
5. gulp
6. bower

Сборка:
1. npm install gulp
2. bower install
3. gulp bower
4. gulp css (gulp watch:css)
5. gulp build
